mrv2 v1.4.0
=======

mrv2 is an open source professional player and review tool for vfx, animation and computer graphics for Windows, Linux and macOS.

It ships now in two versions:
   - mrv2 with OpenGL backend
   - vmrv2 with Vulkan backend (commercial)

This version consolidates and fixes a number of issues on the Vulkan backend.
It adds saving of HDR frame metadata in movies.  Fixes Prores and DNxHR profiles.  Fixes crashes with some .gifs.  Fixes some Wayland crashes and hangs on the Vulkan backend.  Adds international characters support to Vulkan's Text annotations.  Many other Vulkan fixes and improvements.

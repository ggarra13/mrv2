mrv2 v1.4.1
=======

mrv2 and vmrv2 are an open source professional players and review tools for vfx, animation and computer graphics for Windows, Linux and macOS.

It ships now in two compiled versions:
   - mrv2 with OpenGL backend  (free up to version 1.4.0 -
     	       	      	        donationware afterwards)
   - vmrv2 with Vulkan backend (donationware)

Updates OpenEXR to v3.4, FFmpeg to v8.0, fixes a number of Vulkan issues,
adds Vulkan instructions for the different distros, adds the revolutionary
Voice Annotations feature, makes UI more responsive while loading clips from
the command-line.  Adds an option to not load Python, for even faster start
ups.

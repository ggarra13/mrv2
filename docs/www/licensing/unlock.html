Unlocking features of mrv2/vmrv2

Node-Locked Licenses (1 year)

  • Solo (USD $25) → Drawing, storing and saving text and graphical annotations
  • Standard (USD $50) → Solo + Python with its Editor
  • Edit (USD $100) → Standard + Edit Video support + Saving .otio files
  • Pro (USD $150) → Edit + Voice Annotations and URL Link / File Annotations (eventually AI prompt support)

Transferable License

  • Pro+ (USD $300) → All of Pro, but transferable between machines (license ownership retained)

Floating Licenses

  • Not locked to any machine, but to your facility
  • Allows multiple users under the same plan
  • Minimum cost equivalent to 5 licenses of the selected tier

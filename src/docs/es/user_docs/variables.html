Variables de Entorno

EDITOR - Editor a usar en el panel Python. Por defecto: vim. FLTK_BACKEND - Sólo Linux: seteado a x11, pasa Wayland a usar XWayland. HOME - Si está seteado, donde ubicar el directorio .filmaura,

licencia y seteos de preferencis que serán grabados.

LANGUAGE - Languaje a ser usado en la interfaz (Usualmente seteado por

el OS).

OCIO - Dónde se ubica el archivo config de OpenColorIO. MRV2_STUDIOPATH - Variable global para ubicar los seteos de preferencia del

estudio (antes que los del usuario).

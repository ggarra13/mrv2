HDR

mrv2 soporta HDR (High Dynamic Range) internamente pero actualmente no puede mostralo en un monitor HDR ya que es una limitación de OpenGL.

Lo que se puede hacer es usar un SDK de salida y un dispositivo conectado a un monitor externo. Si usa NDI® puede mostrar el contenido en NDI® Tools en un monitor computador capaz de HDR.

NDI®

NDI® usa su LAN (su red local) para transmitir el video y puede ser usado para las revisiones de contenido entre los artistas. Actualmente, el Normal NDI SDK está limitado a 8-bits, no HDR. Esperamos confirmación si el Advanced NDI SDK puede ser distribuído gratuitamente.

Si quiere ver NDI® con HDR en un monitor de computadora con el Advanced SDK, puede usar NDI Tools (en macOS o Windows solamente), los cuales pueden ser descargados de:

https://ndi.video/tools/

HDR

mrv2 en OpenGL soporta HDR (High Dynamic Range) internamente pero actualmente no puede mostralo en un monitor HDR ya que es una limitación de OpenGL.

mrv2 en Vulkan suporta HDR (High Dynamic Range) nativamente, si su monitor y su compositor Wayland en Linux es moderno como GNOME 48.

Lo que se puede hacer además es usar un SDK de salida y un dispositivo conectado a un monitor externo. Si usa NDI® puede mostrar el contenido en NDI® Tools en un monitor del computador capaz de HDR.

NDI®

NDI® usa su LAN (su red local) para transmitir el video y puede ser usado para las revisiones de contenido entre los artistas.

Si quiere ver NDI® con HDR en un monitor de computadora con el Advanced SDK, puede usar NDI Tools (en macOS o Windows solamente), los cuales pueden ser descargados de:

https://ndi.video/tools/

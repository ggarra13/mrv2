Environment variables

EDITOR - Editor to use as external editor in the Python Panel.

Default: vim.

FLTK_BACKEND - Linux only: if set to x11, overrides Wayland to use XWayland. HOME - if set, where the .filmaura directory, license and preference

settings will be saved.

LANGUAGE - Language to used in the UI (Usually set by the OS or mrv2

preferences).

OCIO - OCIO override for OpenColorIO’s config file. MRV2_STUDIOPATH - global variable where to look up preferences settings first

for a studio setting (before the user settings).

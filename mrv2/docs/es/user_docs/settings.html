Seteos

El Panel de Seteos (Tecla por Defecto: F9) permite principalmente configurar el comportamiento del Cache de mrv2. En él, puede controlar cuánta memoria en Gb usa el cache o, si setea el control de Gigabytes a 0, en vez de usar Gigabytes, el seteo de Adelante o Atrás controlará el cache en segundos.

El seteo por defecto de Gigabytes es usar la mitad de tu RAM disponible.
